### Summary
*Downloads Resources over HTTP*<br><br>Upgrade to version 1.3.1 or greater.
### Details
broccoli-closure is a Closure compiler plugin for Broccoli.  broccoli-closure downloads binary resources over HTTP, which leaves it vulnerable to MITM attacks.  It may be possible to cause remote code execution (RCE) by swapping out the requested binary with an attacker controlled binary if the attacker is on the network or positioned in between the user and the remote server.
<br><br>• Affected Versions: <1.3.1
<br>• Patched Versions: >=1.3.1
